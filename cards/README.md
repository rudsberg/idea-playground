# Speed Draw

A self-contained, single-file card game. Draw 20 cards from a shuffled
standard 52-card poker deck as fast as you can — the clock starts on your
first card and stops on the 20th. Your best time is saved in the browser.

## Run locally

Open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000/cards/
```

No build step, no dependencies.
